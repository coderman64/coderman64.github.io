Hey! I'm Coderman64!

I make programs, games, and all sorts of computer experiments. Some of these I put on Github so other people can see them, use them, and experiment with them.

That being said, some of my repos are purely experimental, and I have not yet gone through and cleaned out all of the repos that are empty/virtually empty.

The following repos I have in a fairly working state:

Games

  • The Path - an html5 game that works on mobile devices
  • The Block - a basic mobile platformer that does not use the HTML5 canvas

Web Experiments

  • Navigation App Attempt - an attempt to build a google-maps-style html navigation app using only open-source alternatives like OpenStreetMap
  • webWindow - a javascript package that can put any html element into its own window
  • web synth - an html5 music sequencer, currently a work in progress

Other

  • Cyntacs - a small, lightweight text editing program with features like syntax highlighting and line numbering
  • screen recorder - a python GUI that uses ffmpeg's gdigrab feature to record the screen in windows
  • Browsi - a simple (extremely experimental) web browser built in python using the Tk text widget. Likeliness that the wepage will be hopelessly mangled: 98%
